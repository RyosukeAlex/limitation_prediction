# LimTrends: 科学論文の「研究の限界」を分析するためのLLMを活用した時系列トレンド分析およびトピックモデリング

## 概要 (Description):
従来の単年での分析とは異なり、**LimTrends** は複数年にわたる科学論文の「研究の限界（Limitation）」セクションの**長期的なトレンド分析（時系列分析）**に焦点を当てています。
大規模言語モデル（LLM）と動的トピックモデリング（Temporal BERTopicなど）を統合することで、本プロジェクトは研究上の課題が時間とともにどのように進化、出現、あるいは解消していくかを追跡します。

本研究には、時系列データの前処理、プロンプトエンジニアリング、LLMのファインチューニング、動的モデリングのための各種LLMとBERTopicの統合、そして学術的な課題の年ごとの変遷を理解するための首尾一貫したテキスト要約が含まれています。
